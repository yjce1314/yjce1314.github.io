hexo美化 live2d右下角添加猫咪并改模
发表于 2020-02-11| 更新于 2020-03-17

最近玩这个Hexo博客有点上瘾，看了很多美化方案，现在是安装了 butterfly 这个主题， live2d 插件右下角加了个白猫。这个白猫嘛就是大锤，由于猫是白色而且是鸳鸯眼，于是有了改模这步。具体过程如下：

1.安装模块:

npm install --save hexo-helper-live2d

2.安装模型：

npm install live2d-widget-model-tororo

这里有很多模型可供选择，我只知道其中几种，想知道其他是什么可以安装一下npm install 模型代码
